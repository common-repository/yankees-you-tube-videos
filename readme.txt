== Description ==

Ever wondered how to show Yankees videos to your site visitors?? Here is an easy way to do that... Use this plugin and sit back and relax.


== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload 'yankees' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto Widgets and add this plug in in the sidebar and you are all set.
